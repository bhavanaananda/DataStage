ADMIRAL data file access from Linux

ADMIRAL files can be accessed from Linux as a network share:

If using GNOME, use the "Places > Connect to server ..." menu command. A popup window will appear. From the "Service type" drop-down menu, select "Windows share". Under "Server" enter %{HOSTFULLNAME} and under "Share" enter data. Under "Username" enter username and under "Domain" enter %{WORKGROUP} and then click on "Connect". Another popup window will then appear, into which the password for username should be entered.

Once connected, a user will be able to read/write files in their own private area, and also to read all files in the shared and collab areas.

For advanced users, it is also possible to mount the filesystem directly, using the command:

mount.cifs //${HOSTNAME}/data mountpoint -o rw,user=username,password=userpass,nounix,forcedirectio
where mountpoint has already been created.
