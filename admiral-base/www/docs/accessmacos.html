ADMIRAL data file access from MacOS

ADMIRAL data files may be accessed from MacOS as a Windows network share:

smb://%{HOSTFULLNAME}/data
A username and password will be required to complete a connection.

To connect a Windows share from MacOS, use the Finder "Go > Connect to server..." menu command (or COMMAND-K shortcut), and enter

smb://username@%{HOSTFULLNAME}/data as the server address.
The system will then prompt for the specified user's password.

Once connected, a user will be able to read/write files in their own private area, and also to read all files in the shared and collab areas.
