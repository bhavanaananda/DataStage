ADMIRAL data file web browser access

Selected ADMIRAL data files may be accessed using a web browser, using one of the following forms of URI: 

	http://%{HOSTNAME}.zoo.ox.ac.uk/data/private/username/...
	http://%{HOSTNAME}.zoo.ox.ac.uk/data/shared/username/...
	http://%{HOSTNAME}.zoo.ox.ac.uk/data/collab/username/...
A username and password will be required to access the information. For files in a user's private area, that particular user's credentials must be used.

For files in the shared area, then any research group member's credentials can be used and for collab files any group member's or recognized collaborator's credentials may be used.
